HTML5

La Estructura

HTML es el esqueleto de cualquier sitio web. Define párrafos, títulos e imágenes.

<main>
  <h1>Hola Mundo</h1>
  <p>Soy Chizu</p>
</main>
2

Etiquetas Clave

<div>: Contenedor genérico.
<a>: Enlaces o hipervínculos.
<img>: Para mostrar imágenes.

Tip: HTML no es un lenguaje de programación, es de marcado.

3

JavaScript

Interactividad

Es el lenguaje que le da vida a la web. Permite hacer cálculos y animaciones.

let nombre = "Chizu";
console.log("Hola " + nombre);

function saludar() {
  alert("¡Bienvenido!");
}
4

Variables y Tipos

Usamos let para variables y const para valores fijos.

JS es "débilmente tipado", lo que significa que es muy flexible con los datos.

5

C++

Poder y Rendimiento

Usado en videojuegos y sistemas operativos por su alta velocidad.

#include <iostream>
using namespace std;

int main() {
  cout << "Hola C++";
  return 0;
}
6

Conceptos Pro

Punteros: Manejo directo de la memoria RAM.
POO: Programación Orientada a Objetos.

Es más difícil de aprender, pero es el "padre" de muchos lenguajes.
